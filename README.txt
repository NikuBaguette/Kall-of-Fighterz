Player 1 : 
    q to move left
    d to move right
    z to jump
    u and i to attack

Player 2 : 
    left arrow to move left
    right arrow to move right
    up arrow to jump
    nump 4 and nump5 to attack

To start :
    You need Pygame, type 'python3 -m pip install -U pygame --user' in your cmd
